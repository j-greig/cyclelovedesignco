## Setup

1. Install Gulp globally via Terminal:  
`npm install -g gulp`

2. Install Xcode command line tools:  
`xcode-select --install`

3. Install Jekyll Gem  
`sudo gem install jekyll`

4. Install bundler  
`sudo gem install bundler`
